作为一个C语言初学者，这个项目是我的第一次尝试，如果有任何不规范的地方，还请原谅。
本井字棋项目完全基于C语言开发，其中判定逻辑均为本人独立设计。
目前的版本没有图形化界面，每次输入后需要按Enter键。

功能特性
- 双人对战模式（玩家1：`0`，玩家2：`x`）
- 人机对战模式（可选择先手）
- AI 能自动防守玩家的直接威胁
- 完善的输入验证（防越界、防重复落子）
- 实时棋盘显示
- 胜负与平局判定

Windows环境可直接下载exe文件运行，其他环境需要自行编译。

操作说明
- 启动后：
- 按 a 并回车进入双人对战
- 按 b 并回车进入人机对战
- 按 q 并回车退出

落子时输入数字1-9并回车，对应棋盘位置：

```
1 | 2 | 3 
--+---+---
4 | 5 | 6 
--+---+---
7 | 8 | 9 
```

人机对战会询问你是否先手，输入 Y 或 N 选择。
